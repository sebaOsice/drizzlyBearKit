# Setup the drizzlyBear

Clone this repositiory from your console to a destinated directory

```bash 
cd YOUR-WORKDIR
mkdir YOUR_PROJECT_DIR
git clone https://github.com/stereoIII6/drizzlyBearKit.git)
```

```bash
cd YOU_PROJECT_DIR
npm i
```

```bash
cd app
npm i
```

```bash
cd ..
truffle develop
compile
migrate 
.exit 
```

```bash
cd app
npm start
```
